# 專案介紹

## 專案概述

這是一個為了某人生日建立的小遊戲。這是一個僅支援手機畫面的關卡故事性 APP。

遊戲以**今年去法國打工換宿的冒險**為主線故事，玩家將跟隨主角的腳步，體驗在法國打工換宿期間的各種冒險與挑戰。從抵達法國的那一刻開始，到在異國他鄉的日常生活、工作體驗、文化衝擊，以及各種意想不到的際遇，每個選擇都將影響故事的發展方向。

## 功能特色

- **手機優先設計**：專為手機畫面優化，提供最佳的行動裝置體驗
- **主線故事**：以法國打工換宿的冒險為主軸，呈現真實且豐富的異國生活體驗
- **關卡式故事**：採用關卡式設計，每個關卡都有獨特的故事內容，對應冒險中的不同階段
- **互動式選擇**：每個關卡會有一些選項，玩家可以做出不同的選擇，體驗不同的冒險路線
- **分支劇情**：不同的選項會導向不同的關卡，創造多樣化的遊戲體驗和結局
- **豐富視覺內容**：每個關卡都會搭配打工換宿、國外生活的真實圖片或影片，讓玩家更有臨場感，彷彿親身經歷這段冒險旅程

## 技術棧

- **前端框架**：Vue 3
- **建置工具**：Vite
- **樣式框架**：Tailwind CSS
- **Node 版本**：Node.js 24

## 專案結構

專案使用 Vite + Vue 3 建立，並整合 Tailwind CSS 進行樣式設計。
